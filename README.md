# Guía Rápida de Metodologías de Desarrollo de Software

Este script en Bash proporciona una guía rápida interactiva sobre metodologías de desarrollo de software, incluyendo tanto metodologías ágiles como tradicionales.

## Uso

Para ejecutar el script, simplemente ejecuta el archivo bash

## Menús Disponibles

El script presenta los siguientes menús:

### Menú Principal

- `-a`: Metodologías Ágiles
- `-t`: Metodologías Tradicionales

### Menú de Metodologías Ágiles

- `1`: SCRUM
- `2`: X.P.
- `3`: Kanban
- `4`: Crystal

### Menú de Metodologías Tradicionales

- `1`: Cascada
- `2`: Espiral
- `3`: Modelo V

## Funcionalidades

El script proporciona las siguientes funcionalidades para cada metodología seleccionada:

- Agregar información
- Buscar información
- Eliminar información
- Leer base de información
- Volver al menú anterior
- Salir
